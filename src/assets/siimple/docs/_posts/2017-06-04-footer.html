--- layout: "documentation" category: "layout" title: "Footer" description: "A basic component for displaying content in the bottom of your page" keywords: "foot,footer,content,bottom" ---
To create a footer, simply add the class siimple-footer to a div:
Made with love by siimple 
<div class="siimple-footer" align="center">
    Made with love by <strong>siimple</strong>
</div>
Sizing
Change the size of the footer component adding the classg siimple-footer--[SIZE]
Made with love by siimple 
<div class="siimple-footer siimple-footer--medium" align="center">
    Made with love by <strong>siimple</strong>
</div>
Colored footer
Add siimple-footer--[COLOR] to change the color of the footer. Check available colors in the theming section.
Made with love by siimple 
<div class="siimple-footer siimple-footer--dark" align="center">
    Made with love by <strong>siimple</strong>
</div>
Legacy colors API for the footer component will be removed in a future major release. Please upgrade to the new colors API introduced in v3.1.0.
